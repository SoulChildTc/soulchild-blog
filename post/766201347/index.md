# 阿德勒心理学：原因论 vs 目的论


<!--more-->
 
### 一、 生活中常见的原因论表述
 
1. 我性格内向、天生不爱说话，所以不管是公司团建还是朋友聚会，我都**躲在角落不参与**，甚至提前找借口离场。
2. 我天生慢热、不擅长快速熟络，所以和相亲对象聊天时，我都**不敢主动找话题**，只等着对方开启对话。
3. 我小时候学画画，每次画完都被爸妈指责“画得太差、浪费时间”，所以现在不管学新技能还是接手新工作，只要初期做得不顺畅，我就**立刻退缩放弃**，连再试一次的勇气都没有。
4. 我刚入职场时提过一个优化方案，被同事当众反驳说不切实际，所以现在开会讨论，我都**不敢表达不同观点**，只跟着别人的意见附和。
 
### 二、 阿德勒目的论核心观点
 
不是过去的经历或天生的性格决定现在的状态，而是为了逃避当下的问题，主动拿这些当借口。
 
### 三、 目的论视角拆解常见表述
 
1. 以性格内向归咎回避社交 → 隐性目的是逃避和陌生人打交道的局促，用“内向”的标签回避社交中可能出现的尴尬，避免自己陷入手足无措的境地。
2. 以天生慢热归咎被动聊天 → 隐性目的是逃避主动找话题却得不到回应的挫败感，用“慢热”当挡箭牌，避免承担“聊不好”的责任。
3. 以小时候学画被指责归咎遇事放弃 → 隐性目的是逃避再次被否定、被指责的尴尬，借过往的负面评价回避投入精力却得不到认可的过程，防止陷入自我否定的内耗。
4. 以方案被反驳归咎不敢发言 → 隐性目的是逃避当众被质疑的窘迫，用过去的挫败经历回避表达观点可能带来的争议，维持“不犯错”的形象。
 
### 四、 总结
 
1. 过去的经历、天生的性格都是客观存在，但不会直接决定人生走向。
2. 真正困住人的，是选择用这些客观因素，为自己的不敢改变找理由。
3. 切换目的即可改变行为，将“避免聊砸”转为“哪怕聊一句也行”，就能主动开启对话。
4. 感冒发烧时，医生纠结你穿得太薄或淋了雨毫无意义，关键是及时治疗退烧。
5. 如果我们一直依赖原因论，就会永远止步不前。

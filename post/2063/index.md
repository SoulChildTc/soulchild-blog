# python3环境安装

<!--more-->
```bash
yum install -y openssl-devel bzip2-devel expat-devel gdbm-devel readline-devel

wget https://www.python.org/ftp/python/3.6.4/Python-3.6.4.tgz

tar -zxvf Python-3.6.4.tgz

cd Python-3.6.4

./configure

make

make install
```

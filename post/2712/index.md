# git设置提交者信息

<!--more-->
git配置的三个级别
- `--local`: 仓库级(仅针对某个git仓库`.git/config`)
- `--global`: 用户级(系统用户的配置`~/.gitconfig`)
- `--system`: 系统级(git级配置`/private/etc/gitconfig`)


修改信息
```bash
git config --local user.name 'SoulChild'
git config --local user.email '742899387@qq.com'
```

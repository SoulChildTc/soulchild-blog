# yum安装ffmpeg

<!--more-->
```bash
yum install http://li.nux.ro/download/nux/dextop/el7/x86_64/nux-dextop-release-0-5.el7.nux.noarch.rpm
rpm --import http://li.nux.ro/download/nux/RPM-GPG-KEY-nux.ro
yum install -y ffmpeg-2.8.15-2.el7.nux
ffmpeg -version
```

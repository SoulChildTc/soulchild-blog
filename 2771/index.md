# Screen常用命令

<!--more-->
```bash
# 创建新会话窗口
screen

# 创建新会话窗口-带名称
screen -S s1

# 离开会话窗口
Ctrl + a, d

# 查看所有会话窗口
screen -ls

# 进入已存在的会话窗口
screen -r (id or name)

# 断开指定的会话窗口
screen -d (id or name)

# 杀掉当前会话窗口(杀掉会话窗口,窗口将不存在)
Ctrl + a, k
或
退出终端Ctrl + d
```

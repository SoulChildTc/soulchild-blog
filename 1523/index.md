# shell颜色

<!--more-->
```bash
export PS1="\[\e[37m\][\[\e[32m\]\u\[\e[37m\]@MBP \[\e[36m\]\W\[\e[0m\]]\\$ "
```

说明：
字体颜色 字体背景颜色
30：黑
31：红
32：绿
33：黄
34：蓝色
35：紫色
36：深绿
37：白色

字体背景颜色
40：黑
41：深红
42：绿
43：黄色
44：蓝色
45：紫色
46：深绿
47：白色

显示方式
0：终端默认设置
1：高亮显示
4：下划线
5：闪烁
7：反白显示
8：隐藏
格式：
\033[1;31;40m # 1 是显示方式，可选。31 是字体颜色。40m 是字体背景颜色。\033可以用\e代替
\033[0m # 恢复终端默认颜色，即取消颜色设置。
